# 安装

前提条件

由于本组件库是基于 view-ui-plusopen in new window 和 element-plusopen in new window 开发的，所以要先引入它们

# 环境支持

Edge ≥ 79Firefox ≥ 78Chrome ≥ 64Safari ≥ 12

# 使用包管理器

$ yarn add @web-basic-doc/components

$ npm install @web-basic-doc/components --save

$ pnpm install @web-basic-doc/components
